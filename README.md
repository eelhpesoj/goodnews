# 광진교회 단일 페이지

GitHub Pages에 바로 올릴 수 있는 정적 웹페이지입니다.

## 수정할 부분

- `index.html`에서 각 `<a href="...">` 값을 실제 링크로 변경하세요.
- `assets/` 폴더의 SVG 파일을 실제 이미지 파일로 교체해도 됩니다.
  - `logo.svg`
  - `youth.svg`
  - `choir.svg`
  - `tscw.svg`
  - `map.svg`

## GitHub Pages 배포

1. GitHub에 새 Repository 생성
2. 이 폴더 안의 파일 전체 업로드
3. Repository Settings → Pages
4. Branch를 `main`, Folder를 `/root`로 설정
5. Save
